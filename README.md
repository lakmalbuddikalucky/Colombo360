# Colombo360
This project is a location based social network for 360 panoramas. 

Technology used:
AngularJS,
PHP7,
HTML5,
CSS,
MySQL

Things to be done

Lakmal - News feed, Profiles

Madhawa - Explore (the map) and User management

# How to user the code
*Put the directory in the www folder of Uni server

*Run the uni server

*Import the Database to the DB server with the provided SQL file
